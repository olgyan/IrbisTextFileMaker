Irbis Text File Maker ver. alpha

Программа разбирает библиографические описания статей и одночастных ресурсов.


--={ Как пользоваться }=--

- скопируйте библиографическое описание, например ссылку для цитирования из eLibrary
- нажмите "Вставить" - описание появится в текстовом поле
- убедитесь, что все предписанные знаки препинания на месте и что до и после них стоят пробелы в соответствии с ГОСТ 7.0.100-2018, иначе ссылка будет разобрана некорректно
- нажмите "Разобрать"
- повторите со следующими БО
- чтобы разобрать целый список БО сразу, нажмите "Разобрать из файла" и укажите путь к файлу со списком, файл должен быть в формате .txt
- когда все ссылки будут разобраны, нажмите "Сохранить в текст. файл ИРБИС"
- если нужно создать новые записи, импортируйте записи из файла в Ирбис в режиме "слияние"
- если нужно дополнить существующие записи, создайте на каждую запись отдельный файл и выполните в Ирбисе "Вставить из импорт. записи"


--={ Ограничения программы }=--

- Ctrl-V не работает, вся вставка из буфера делается нажатием кнопки рядом с полем
- заменяет все "ё" на "е"
- плохо обрабатывает многословные функции (типа "под редакцией заслуженного деятеля науки РФ, доктора экономических наук, профессора Фамилии И. Б.").
- не отличает коллектива от автора (считает всё, что написано в сведениях об ответственности с большой буквы, фамилией, что с маленькой - функцией). Коллектив надо будет перенести из 961 в 962 вручную.
- если в свед. об отв. указан только редактор, кладет его в поле 701, а не 702, надо перебросить вручную
- "под ред. И. Б. Фамилии" так в род. падеже в поле и кладет, надо менять падеж вручную
- свойство Ирбиса: если в import.txt будет несколько одинаковых записей, положит их в одну запись друг поверх друга


--={ Файлы }=--

app.py - основной файл программы
contents.py - делает содержание (пока не работает)
grnti.py - содержит соответствия предметных рубрик и рубрик ГРНТИ
config.py - файл настроек
profs.py - содержит список преподавателей
readme.txt - данный файл
rusmarc.py - содержит список кодов для функций персон (редакторы и пр.)
